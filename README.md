# Rodan-Fields
Rodan &amp; Fields is an American manufacturer and multi-level marketing company specializing in skincare products. We build clone of their website during Unit-II construct week with a team of 6 members.

## Abstract

This project is a one to one clone of the online cosmetics website of "Rodan+Fields" which was done as a part of Masai School Construct Week #2. The project includes the Landing page, Login and Signup page, Product pages as well as Best Seller page. The project is truly interactive and functional with features such as Login , Sorting product based on keys, and Filter based on categories, Cart feature and Payment checkout.

  ## Core Features
- Login and Signup Pages
- Home Page
- Fully Functional Interactive Navigation Bar
- Product Section based on Categories
- Sort Feature
- Search Option
- Cart Feature
- Payment Checkout


## Technologies Used

- HyperText Markup Language
- Cascading Style Sheets
- Node.js
- Local Storage
